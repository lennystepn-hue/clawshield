# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ClawShield, please report it responsibly.

**Do NOT open a public issue.**

Email: **security@openclaw.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to release a fix within 7 days for critical issues.

## Scope

ClawShield is a security auditing tool. Vulnerabilities in ClawShield itself (e.g., command injection via skill paths, privilege escalation) are in scope. Issues found *by* ClawShield on your system are not bugs in ClawShield.
